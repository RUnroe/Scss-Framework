HTML Elements

Containers

A class applicable to sections and divs. Two different versions: .container and .bordered-container.

I am a container
I am a bordered container
<div class="container">
  I am a container
</div>
<div class="bordered-container">
  I am a bordered container
</div>

Input Fields

By default, text input fields take up 100% width. Use .col-# classes to change the size in a responsive manner (shown below). Submit inputs can use button classes as shown below. 

<input type="text" placeholder="Name" />
<input type="text" placeholder="Name" class="col-2" />
<div class="row content-horizontal-align-center">
  <input type="submit" class="btn btn-dark" />
</div>

Images

By default, <img> tags will take up the max width of their parent and scale with their parent.

<img src="https://via.placeholder.com/150" />
<img src="https://via.placeholder.com/150" />
<img src="https://via.placeholder.com/150" />

Buttons

Buttons

Buttons do not have their own default styling. Use the .btn class to get basic button styling. This class can be used on button, div, and anchor tags.

<button class="btn btn-primary">
  Primary
</button>
<button class="btn btn-secondary">
  Secondary
</button>
<button class="btn btn-success">
  Success
</button>
<button class="btn btn-danger">
  Danger
</button>
<button class="btn btn-dark">
  Dark Theme
</button>
<button class="btn btn-light">
  Light Theme
</button>

Button Group

Buttons do not have their own default styling. Use the .btn class to get basic button styling. This class can be used on button, div, and anchor tags.

<div class="btn-group">   
  <button class="btn btn-primary">
    Left
  </button>
  <button class="btn btn-primary">
    Center
  </button>
  <button class="btn btn-primary">
    Right
  </button>
</div>

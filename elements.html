HTML Elements

Containers

A class applicable to sections and divs. Two different versions: .container and .bordered-container.

I am a container
I am a bordered container
<div class="container">
  I am a container
</div>
<div class="bordered-container">
  I am a bordered container
</div>

Input Fields

By default, text input fields take up 100% width. Use .col-# classes to change the size in a responsive manner (shown below). Submit inputs can use button classes as shown below. 

<input type="text" placeholder="Name" />
<input type="text" placeholder="Name" class="col-2" />
<div class="row content-horizontal-align-center">
  <input type="submit" class="btn btn-dark" />
</div>

Images

By default, <img> tags will take up the max width of their parent and scale with their parent.

<img src="https://via.placeholder.com/150" />
<img src="https://via.placeholder.com/150" />
<img src="https://via.placeholder.com/150" />

Modals

Create a modal by applying the .modal class to a <div> tag. Create a button and give it a data-target attribute set to the id of the modal to link them. Modals are easily customizable and there can be more than one per page.

Modal Header

✖

Modal Text Goes Here

Dark Modal Header

✖

Put anything you want to say in the modal right here

<button class="btn btn-light" data-toggle="modal" data-target="basicModal">
  Open Basic Modal
</button> 

<div id="basicModal" class="modal">
  <div class="modal-content">
    <div class="modal-header">
      <h1>
        Modal Header
      </h1>
      <span class="modal-close">&#10006;</span>
    </div>
    <div class="modal-body">
      <p>
        Modal Text Goes Here
      </p>
    </div> 
  </div>
</div>

<button class="btn btn-dark" data-toggle="modal" data-target="darkModal">
  Open Dark Modal
</button> 

<div id="darkModal" class="modal">
  <div class="modal-content">
    <div class="modal-header dark">
      <h1>
        Dark Modal Header
      </h1>
      <span class="modal-close">&#10006;</span>
    </div>
    <div class="modal-body">
      <p>
        Put anything you want to say in the modal right here
      </p>
    </div> 
  </div>
</div>

Default Alert

An alert is a small highlighted block of text. It can be created by applying .alert and a color type to a <div>.

Closable Alert

This alert works the same as the default but will allow the user to close it if they wish. In order to accomplish this, add .alert-closable to the <div>.

Primary Alert
Light Alert
Dark Alert
<div class="alert alert-closable alert-primary">
  Primary Alert
</div>
<div class="alert alert-closable alert-light">
  Light Alert
</div>
<div class="alert alert-closable alert-dark">
  Dark Alert
</div>

Buttons

Buttons

Buttons do not have their own default styling. Use the .btn class to get basic button styling. This class can be used on button, div, and anchor tags.

<button class="btn btn-primary">
  Primary
</button>
<button class="btn btn-secondary">
  Secondary
</button>
<button class="btn btn-success">
  Success
</button>
<button class="btn btn-danger">
  Danger
</button>
<button class="btn btn-dark">
  Dark Theme
</button>
<button class="btn btn-light">
  Light Theme
</button>

Button Group

Buttons do not have their own default styling. Use the .btn class to get basic button styling. This class can be used on button, div, and anchor tags.

<div class="btn-group">   
  <button class="btn btn-primary">
    Left
  </button>
  <button class="btn btn-primary">
    Center
  </button>
  <button class="btn btn-primary">
    Right
  </button>
</div>

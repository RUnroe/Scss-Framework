Grid

This framework uses flex to create a grid like system with 6 vertical splits (3 on mobile).

Equal Width

Elements with the class .col contained in a .row will take up equal widths.

</div>

Navbar

Top and side navigation bars. Top nav uses the <nav> tag. Side nav bar uses sections or divs with specific classes.

Top Nav

Top nav bars come with four different styles: .navbar-primary .navbar-secondary .navbar-light and .navbar-dark which get applied directly to the <nav> tag.

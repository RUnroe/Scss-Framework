Layout

Grid

This framework uses flex to create a grid like system with 6 vertical splits (3 on mobile).

Equal Width

Elements with the class .col contained in a .row will take up equal widths.

1 of 2 columns
1 of 2 columns
<div class="row" >
    <div class="col bordered-container">
        1 of 2 columns
    </div>
    <div class="col bordered-container">
        1 of 2 columns
    </div>
</div>

Navbar

Top and side navigation bars. Top nav uses the <nav> tag. Side nav bar uses sections or divs with specific classes.

Top Nav

Top nav bars come with four different styles: .navbar-primary .navbar-secondary .navbar-light and .navbar-dark which get applied directly to the <nav> tag. Add the classes as shown below to make a list of links in the nav bar. the .navbar-brand is an optional addition that that takes the first position in the nav bar. The brand can be text, an icon, or an image.

Header
